## Project description

This project was a part of my schools Go course. We developed the project using Gitea(this was our school's choice of VCS) and that's why the commit history is pretty empty.
I made a web forum with a session based authentication.
### Features
- Register & login
- Post - including associating them with tags
- Comment posts
- Like and dislike posts & comments
- Filter posts by tags

## Run the project

Navigate to project root directory **/forum-authentication**<br>
Run ```go run cmd/forum/main.go```<br>
Go to **http://localhost:8000**

## Stack
Frontend
- JQuery Select2
- Bootstrap 5
- HTML & CSS

Backend
- Golang
- SQLite
